## 🚀 خطوات التشغيل

### 1️⃣ إنشاء قاعدة بيانات Supabase

1. اذهب إلى [supabase.com](https://supabase.com) وأنشئ حساباً
2. أنشئ مشروعاً جديداً
3. انتظر حتى ينتهي إعداد المشروع (دقيقتان تقريباً)

### 2️⃣ إنشاء الجداول

1. في لوحة Supabase، اذهب إلى **SQL Editor**
2. افتح ملف `database/schema.sql` من المشروع
3. انسخ كامل المحتوى والصقه في SQL Editor
4. اضغط **Run** لتنفيذ السكريبت

### 3️⃣ ربط المشروع بـ Supabase

1. في Supabase، اذهب إلى **Project Settings** → **API**
2. انسخ القيمتين التاليتين:
   - **Project URL**
   - **anon / public key**
3. افتح ملف `js/config.js` في المشروع
4. ضع القيمتين مكان:
   ```javascript
   SUPABASE_URL: 'https://YOUR_PROJECT_ID.supabase.co',
   SUPABASE_ANON_KEY: 'YOUR_SUPABASE_ANON_KEY_HERE',
   ```
5. غيّر كلمة مرور الإدارة:
   ```javascript
   ADMIN_PASSWORD: 'كلمة_مرور_جديدة_قوية'
   ```

### 4️⃣ التشغيل المحلي

افتح `index.html` مباشرة في المتصفح، أو شغّل خادمًا محليًا:

```bash
# Python 3
python -m http.server 8000

# أو Node.js
npx serve .
```

ثم افتح `http://localhost:8000`

### 5️⃣ إعداد المحتوى

1. افتح `admin.html` وادخل بكلمة المرور
2. **أضف أيام المسابقة** من قسم "أيام المسابقة"
3. **ارفع قائمة الطلاب** من قسم "الطلاب" (ملف Excel)
4. **ارفع الأسئلة** من قسم "الأسئلة" (ملف Word)
5. **فعّل اليوم** المطلوب
6. شارك رابط `index.html` مع الطلاب

## 🔐 الأمان

- ✅ كلمة مرور الإدارة في `config.js` - **غيّرها قبل النشر**
- ✅ الـ Anon Key آمن للاستخدام في المتصفح (لا تشارك Service Role Key)
- ✅ تم تفعيل Row Level Security (RLS) في قاعدة البيانات
- ⚠️ لإنتاج حقيقي، يُنصح بإضافة طبقة حماية إضافية على لوحة الإدارة

## 🎨 التخصيص

- **الألوان**: عدّل المتغيرات في `css/main.css` تحت `:root`
- **اسم المدرسة**: في `js/config.js` تحت `SCHOOL_NAME`
- **عدد الأسئلة اليومية**: في `js/config.js` تحت `QUESTIONS_PER_DAY`
- **مدة السؤال**: في `js/config.js` تحت `TIME_PER_QUESTION`

## 🛠️ التقنيات المستخدمة

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Supabase (PostgreSQL)
- **Libraries**:
  - [Supabase JS](https://supabase.com/docs/reference/javascript) - الاتصال بقاعدة البيانات
  - [SheetJS (xlsx)](https://sheetjs.com/) - قراءة/كتابة Excel
  - [Mammoth.js](https://github.com/mwilliamson/mammoth.js) - قراءة ملفات Word
- **Fonts**: Tajawal & Cairo (Google Fonts)
