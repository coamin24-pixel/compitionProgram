# مجلد الأصول (Assets)

هذا المجلد مخصص لتخزين:
- 🖼️ صور المدرسة (الشعار)
- 🖼️ صور الأسئلة
- 🎨 الأيقونات والرسومات

## رفع صور الأسئلة

لإضافة صورة لسؤال:

### الخيار 1: استخدام Supabase Storage (موصى به)
1. اذهب لـ Supabase Dashboard
2. اختر **Storage**
3. أنشئ Bucket جديد باسم `question-images` واجعله Public
4. ارفع الصور
5. انسخ الـ Public URL والصقه في حقل "رابط الصورة" بلوحة الإدارة

### الخيار 2: استخدام مجلد assets
1. ضع الصورة في هذا المجلد، مثلاً: `assets/question1.jpg`
2. في لوحة الإدارة، اكتب: `assets/question1.jpg` في حقل رابط الصورة

### الخيار 3: استخدام رابط خارجي
استخدم أي رابط مباشر للصورة من الإنترنت.
